# F-Orkut Payment Gateway

Secure Payment Gateway API menggunakan Express.js + Supabase + OrderKuota Integration.

Dibuat untuk kebutuhan Payment Gateway, QRIS, Transaction Management, dan Integrasi OrderKuota API dengan sistem keamanan modern.

---

# Features

- Express.js REST API
- Supabase Database Integration
- API Key Authentication
- Rate Limiter Protection
- Helmet Security
- CSRF Protection
- CORS Protection
- QRIS Withdraw
- Transaction Management
- OrderKuota Integration
- Webhook System
- Input Validation
- Usage Logs
- AJAX Request Protection

---

# Tech Stack

- Node.js
- Express.js
- Supabase
- Axios
- Express Validator
- Helmet
- CORS
- Rate Limit
- CSRF
